# xinsite v0 plan

## Goal
Build the main framework for Xin's new industry-facing personal website.

## Scope
1. Next.js 16 + Tailwind CSS 4 scaffold
2. Homepage sections: Hero → Selected Work → Technical DNA → Experience → Thinking → Contact
3. Assistant placeholder UI panel + stub `POST /api/chat` route (not implemented)
4. `/research` subpage with all academic publications, profile links, teaching
5. Deployment-friendly defaults (Vercel or static export later)

## Explicit non-goals (v0)
- Real webchat / voice assistant backend
- Voice interaction
- Final polished animations / Framer Motion
- Final content copy polish
- Custom domain setup

## Stack
- Next.js 16 App Router + TypeScript
- Tailwind CSS 4
- Geist fonts
- pnpm

## Deliverables
- Runnable local site (`pnpm dev`)
- Homepage shell with all sections
- `/research` subpage populated from publication data
- Assistant interface placeholder with future API hook
